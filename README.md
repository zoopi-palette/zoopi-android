# 🍀zoopi-android
반려동물 헌혈 관리 서비스 Zoopi 입니다.

## 📚  기술스택

- **Minimum SDK Version** 26

- **Target SDK Version** 32

- **Clean Architecture**  지향

- **Architecture**
    - MVVM, MVI

-   **Language**
    - 100% Kotlin

-   **단위 테스트**
    - JUnit
    - Mockito
    - Espresso

-   **비동기 처리**
    - Coroutine

-   **사용한 라이브러리**
    - Jetpack Compose
    - Room, Retrofit, Gson, Glide, Jetpack AAC, Jetpack Hilt
